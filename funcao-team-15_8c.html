void Transposta ( int  V[])

Esta função cria a matriz transposta de uma matriz passada com parâmetro, neste caso a função está limitada a matrizes 18x18. Todos os valores da matriz passada como parâmetro têm de ser valores inteiros.

Para a execução desta funçao é necessário a execução prévia do ponto anterior (ponto 3 -> Cálculo e devolução da matriz 18x18 resultante do produto do vetor inicial com o mesmo vetor ordenado por ordem crescente). Após a execução desta função, é devolvida a matriz transposta da matriz passada como parâmetro.

Esta função faz com que seja mostrada no ecrã uma matriz com tamanho 18x18. Os 'if's utilizados na função apenas servem para uma questão de estética, evitando que a matriz fique com um formato irregular. Ou seja, os 'if's apenas controlam o espaçamento entre os valores da matriz.

◆ ValoresMaioresQue2EDivisiveisPor5()

void ValoresMaioresQue2EDivisiveisPor5 ( int  V[])

Esta função, executa um ciclo que percorre o vetor, criado apartir dos valores introduzidos pelo utilizador. Dentro do ciclo são utilizadas duas condições. Na primeira é verificado se um dado valor é superior a 2. Na segunda é verificado se o resto da divisão deste mesmo valor por 5 é igual a 0. Se as condições forem verificadas, os respetivos valores são mostrados no ecrã.

◆ ValoresSimetricos()

void ValoresSimetricos ( int  V[])

Nesta função, primeiramente, todos os valores do vetor são alterados pelo seu simétrico. Como todos os valores são positivos (compreendidos entre 1 e 11), todos os simétricos serão valores negativos. Posteriormente, com a utilização de um ciclo e uma variável auxiliar, os valores são organizados por ordem crescente. Com estes valores é criado um novo vetor.
